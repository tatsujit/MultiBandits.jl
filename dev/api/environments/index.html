環境 (Environments)

MultiBandits.AbstractEnvironment — Type
AbstractEnvironment

全環境の基底抽象型。サブタイプは sample_reward と mean を実装する。

source
MultiBandits.Environment — Type
Environment <: AbstractEnvironment

定常バンディット環境。各腕は固定の報酬分布を持つ。

コンストラクタ

  • Environment(reward_probs::Vector{Float64}) — Bernoulli 分布の成功確率を指定
  • Environment(distributions) — 任意の単変量分布を指定
  • Environment(n_arms::Int) — n_arms 本の腕を等間隔の Bernoulli 確率で生成

例

env = Environment([0.2, 0.5, 0.8])  # 3本腕の Bernoulli バンディット
source
MultiBandits.NonStationaryEnvironment — Type
NonStationaryEnvironment <: AbstractEnvironment

非定常バンディット環境。change_points で指定した試行で報酬分布が切り替わる。

コンストラクタ

NonStationaryEnvironment(initial_probs, change_points, phase_probs)

例

env = NonStationaryEnvironment(
    [0.2, 0.8],       # 初期確率
    [50],              # 50試行目で切替
    [[0.8, 0.2]]       # 切替後の確率
)
