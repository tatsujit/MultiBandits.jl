システム (System)

MultiBandits.Agent — Type
Agent

方策 (AbstractPolicy) と推定器 (AbstractEstimator) を組み合わせたバンディットエージェント。

フィールド

  • policy::AbstractPolicy — 行動選択方策
  • estimator::AbstractEstimator — 行動価値推定器
source
MultiBandits.System — Type
System <: AbstractSystem

エージェント・環境・履歴を束ねたシミュレーションシステム。

フィールド

  • agent::Agent — バンディットエージェント
  • env::AbstractEnvironment — バンディット環境
  • history::AbstractHistory — 試行履歴の記録先
  • rng::AbstractRNG — 乱数生成器

例

system = System(agent, env, History(n_arms, 100); rng=MersenneTwister(42))
run!(system, 100)
source
MultiBandits.run! — Function
run!(system::System, trials::Int, verbose::Bool=false)

trials 回の試行を実行する。各試行で step! を呼び出す。

source
MultiBandits.step! — Function
step!(system::System, trial::Int, verbose::Bool=false)

1ステップ (t → t+1) を実行する。行動選択 → 報酬サンプリング → 履歴記録 → 推定器更新 の順。

source
MultiBandits.AbstractHistory — Type
AbstractHistory

全履歴型の基底抽象型。シミュレーション中の行動・報酬・期待値を記録する。

source
MultiBandits.History — Type
History <: AbstractHistory

定常環境用の履歴。各試行の行動・報酬と、環境の期待値を記録する。

フィールド

  • n_arms::Int — 腕の数
  • trials::Int — 試行数
  • actions::Vector{Int} — 各試行で選択された行動
  • expectations::Vector{Float64} — 環境の期待報酬（定常なので共通）
  • rewards::Vector{Float64} — 各試行で得た報酬
source
MultiBandits.EstimatorHistory — Type
EstimatorHistory <: AbstractHistory

CognitiveEstimator の内部状態（Q, V, W, C, N, 選択確率）を毎試行記録する拡張履歴。

フィールド

  • Qss::Matrix{Float64} — 各試行 × 各腕の Q 値
  • Vss::Matrix{Float64} — 各試行 × 各腕の効用値 V
  • Wss::Matrix{Float64} — 各試行 × 各腕の総合行動価値 W
  • Css::Matrix{Float64} — 各試行 × 各腕の固執性 C
  • Pss::Matrix{Float64} — 各試行 × 各腕の選択確率
source
MultiBandits.LAHistory — Type
LAHistory <: AbstractHistory

Limited Availability (制限付き腕) 環境用の履歴。各試行で利用可能な腕のサブセットも記録する。
